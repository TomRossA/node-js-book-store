# node-js-book-store

Simple RESTful Node.JS+Express server to run react-redux-book-store app.

### Installing

To install application run

```
npm install
```

To start application

```
npm start
```

## Built With

* [Node.js](https://nodejs.org/en/)
* [Express.js](https://expressjs.com) - Web framework
